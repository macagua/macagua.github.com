Installers

I created some configurations for Plone demonstrations in different use case, the following are:

  • Buildout Wordpress Demo, the idea with this buildout configuration is show install a Non Python Application with Buildout.
  • Plone Demo Installers, the idea with this buildout configuration is show install a Plone latest version with the commons and usefull third-party add-ons Plone's Products.
  • PloneEdu Demo Installers, the idea with this buildout configuration is show install a Plone latest version with the commons and usefull third-party add-ons Plone for education sector.
  • Plone Theme Installers, the idea with this buildout configuration is show install a Plone latest version, with many Plone theme skins, Diazo Themes and usefull Theming tools for layout, adding custom icons and fonts with third-party add-ons theming into Plone.
  • Diazo Theme Installers, the idea with this buildout configuration is show how install a Plone latest version, for integrate with Diazo Theming for a Django application, Joomla, Drupal and Wordpress CMS with Plone in the same GUI. Using diazo as a Diazo theming server and thememapper as a Diazo theming editor external.
  • Plone High Availability Installers, the idea with this buildout configuration is show install a Plone latest version, with a common High Availability Deployment Schema.

The idea is create some website demo public for new users, like a solution's proposal presented by Nejc Zupan with https://github.com/niteoweb/heroku-buildpack-plone.
